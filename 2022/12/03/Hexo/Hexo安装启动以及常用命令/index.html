Hexo 安装启动以及常用命令

2022-12-03 Hexo Hexo 0 评论

准备工作

官网：https://hexo.io/zh-cn/

安装 Hexo

  1. 先要安装 node.js
  2. 安装 Hexo 命令：npm install -g hexo-cli
  3. 配置环境变量：echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile

启动 Hexo

  1. 新建一个空的文件夹，在此目录打开 cmd 命令行
  2. 初始化项目：hexo init
  3. 重新下载未完成的依赖：npm install
  4. 启动 Hexo 服务：hexo server

常用命令

  • 启动 Hexo 服务：hexo server

  • 创建新页面：hexo new page <title>

  • 创建新文章：hexo new post <title>

  • 清除缓存文件：hexo clean

  • 生成静态文件：hexo g

  • 部署到 github：hexo d
