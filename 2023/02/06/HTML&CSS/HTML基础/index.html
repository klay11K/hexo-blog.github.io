HTML 基础

HTML 基础标签

  • html 页面中由一对标签组成：<html></html>，其中 <html> 为开始标签，</html> 为结束标签

  • <head> 标签标识网页头部

    • title 标签表示网页的标题
    • 可以在 meta 标签中使用 charset 属性设置编码方式

  • <br/> 标识换行。<br/> 标签是一个单标签，单标签：开始标签和结束标签是同一个，斜杠放在单词后面

  • <p> 表示段落标签

  • <img> 表示图片标签

    • src 表示图片文件的路径，使用绝对或相对路径都可以
    • width 和 height 表示图片的大小
    • alt 表示图片的提示

  • <h1> ~ <h6> 标题标签

  • <ol> 有序列表，子标签为 <li>

    • start，表示从 * 开始
    • type ，显示的类型：A a i 1(default)

  • <ul> 无序列表，子标签为 <li>

  • <b> 加粗

  • <u> 下划线

  • <i> 斜体

  • <sub> 下标

  • <sup> 上标

  • <hr/> 分割线

  • <a> 超链接

    • href，引用的链接
    • targe，打开的方式
      • _blank，在一个新窗口打开
      • _self，在本窗口打开
      • _parent，在父窗口打开
      • _top，在顶层窗口打开

  • table 表格

    • tr 行

      • align，对齐方式：center，left，right
      • rowsapn，行合并

    • td 列

      • colspan，列合并

    • th 表头列

    • table 的属性

      • border，表格边框的粗细
      • width：表格的宽度
      • cellspacing：单元格间距
      • cellpadding：单元格填充

详见：https://www.w3school.com.cn/html/index.asp

HTML 实体符号

  • &lt; 小于
  • &le; 小于等于
  • &gt; 大于
  • &ge; 大于等于
  • &copy; 版权符

详见：https://www.w3school.com.cn/html/html_entities.asp

Form 表单

  • <form> 表示一个表单标签，action 属性表示要发送请求的地址

  • <input> 表示一个输入标签

    • type = text，表示一个文本框
    • type = password，表示一个密码框
    • type = radio，表示一个单选框
    • type = checkbox，表示一个多选框

  • <select> 是一个下拉标签，其中 <option> 表示一个下拉项

  • <textarea> 表示一个文本域

  • <button> 表示一个按钮

    • type = submit，表示这是一个提交按钮

    • type = rest，表示这是一个重置按钮

    • 普通按钮 <button> 不需要指定属性

  • 表单中的 name 属性对应服务器端的入参字段

详见：https://www.runoob.com/html/html-forms.html
